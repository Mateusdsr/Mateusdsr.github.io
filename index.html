Sobre Mim

Sou estudante do 7º semestre de Ciências da Computação, apaixonado por tecnologia e desenvolvimento de software. Durante minha jornada acadêmica, desenvolvi habilidades sólidas em linguagens de programação como JavaScript e Python, além de experiência prática com HTML, CSS e bancos de dados.

Participei de uma bolsa de estudos na Compass, onde me aprofundei no desenvolvimento com React e trabalhei com meu squad na criação de um aplicativo de e-commerce. Essa experiência proporcionou uma visão prática do uso de tecnologias modernas e metodologias ágeis além de certificações em áreas como Agile (Scrum e Kanban), AWS Cloud Practitioner, Cyber Security, Git e GitHub, JavaScript, React e Redux, React Native, React Testing Library (RTL) e TypeScript.

Estou em constante aprimoramento e busco novas oportunidades para aprender e crescer como desenvolvedor. Meu objetivo é iniciar minha carreira em tecnologia, contribuindo para projetos desafiadores e colaborando com equipes dinâmicas para criar soluções inovadoras.

Estou motivado a me desenvolver como um profissional completo, combinando conhecimentos técnicos com habilidades interpessoais para contribuir para o sucesso de equipes e empresas. Estou pronto para enfrentar novos desafios e continuar evoluindo como desenvolvedor de software.
